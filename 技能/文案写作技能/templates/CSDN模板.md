# CSDN文案模板（优化版）

## 标题公式
```
🎉 【{标签}】{主题} + {核心价值}
——{副标题，一句话说明文章价值}
```

示例：
```
🎉 【2027考研干货】全流程规划 + 公共课书单推荐（超全升级版）
——帮你少走弯路，高效备考，稳稳上岸！
```

## 正文结构

1. 金句开场（吸引注意力）
2. 背景描述（为什么要看这篇文章）
3. 文章价值（用emoji列出包含内容）

## 写作要点

### ✅ 必须做到
1. **结构清晰**：前言 → 分阶段详述 → 书单 → 数据 → 避坑 → 建议 → 福利 → 寄语
2. **表格丰富**：每个章节都用表格整理信息
3. **emoji引导**：每段开头用emoji增强视觉层次
4. **进阶建议**：针对不同读者给出不同方案
5. **口诀/铁律**：用押韵句子帮助记忆
6. **福利详细**：列出具体包含的文件
7. **互动引导**：评论区留言、转发收藏、关注

### ❌ 必须避免
1. 空洞鸡汤，没有实操方法
2. 表格缺失，纯文字堆砌
3. emoji过多，影响阅读
4. 没有分层建议（不同基础/需求）
5. 福利资源描述模糊
