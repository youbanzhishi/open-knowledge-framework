# 文案写作技能 - 使用说明

## 核心能力

本技能支持两种工作模式：

### 模式一：生成新文章
根据资源信息和分享链接，生成各平台文案

### 模式二：润色现有文章
对已有文章进行优化，支持多种润色方向：
- 去AI味：消除AI特征，增加真人感
- 提升标题吸引力：数字量化、场景化、情绪触动
- 优化开头钩子：3秒抓住用户
- 增加互动设计：提升评论率
- 调整结构节奏：段落长度、钩子密度
- 平台适配：根据目标平台调整风格

---

## 快速开始

### 场景一：生成新文章
本技能被网盘变现主理技能调用，传入以下参数：

```json
{
  "resource_name": "考研备考规划",
  "resource_type": "考研资料",
  "share_link": "https://url98.ctfile.com/f/xxx-xxx-xxx",
  "password": "8069",
  "platforms": ["知乎", "小红书", "微博", "CSDN"],
  "style": "专业型"
}
```

### 输出位置
生成的文案存放在：
`./网盘引流项目/待发布文章/YYYY-MM-DD_平台_资源名称.md`

## 核心能力

### 1. 多平台适配
- **知乎**：专业、结构化、表格、深度（1000-1500字）
- **小红书**：emoji、短句、互动引导、评论区引流（300-500字）
- **微博**：简短、话题标签、快速传播（200-400字）
- **CSDN**：技术感、表格、专业术语（1000-1500字）
- **博客园**：程序员向、技术细节（同CSDN）

### 2. AI去机味
- 禁用成语、书面词、套话
- 使用口语化、真实情绪、个人观点
- 通过"真人真话测试"

### 3. 链接植入
- 知乎/CSDN/博客园：描述性链接
- 微博：直接放链接
- 小红书：⚠️ 不能放链接，用评论区引流

---

## 质量标准

### 三要素
1. **有价值**：信息密度高、实用性强
2. **有吸引力**：标题抓人、开头留人、结尾钩子
3. **有真人感**：不端着、有细节、有情绪

### 检测标准
通过"真人真话测试"：
> 这段话像"给所有人看的正确答案"，还是"一个真人在说自己的真话"？
