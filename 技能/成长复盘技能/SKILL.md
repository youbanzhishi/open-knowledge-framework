# 成长复盘技能

## 技能信息
- 加载模式：progressive
- 触发关键词：[复盘, 成长, 周结]
- 摘要：定期复盘+信息一致性检查+工作流优化

## 技能概述
定期复盘总结，检查信息同步一致性，优化改进工作流程，帮助你持续成长。

## 适用场景
- 每周定期复盘
- 发现并修正信息不一致
- 目录结构整理
- 总结经验教训
- 优化工作流程

## 核心功能

### 1. 目录结构整理
检查文件是否在正确位置：
- **自定义技能** → 必须在 `./技能/` 目录下
- **复盘记录** → 必须在 `./复盘记录/` 目录下
- **定时任务配置** → 必须在 `./定时任务/` 目录下

**执行步骤：**
1. 扫描根目录，识别是否有技能目录错放
2. 发现错放的技能目录 → 移动到正确位置
3. **关键：** 移动后搜索所有文件中的旧路径引用 → 更新为新路径
4. 记录移动操作到复盘报告

### 2. 配置一致性检查
自动检查以下文件是否同步：
- `./定时任务/tasks.json` ↔ 实际日程
- `./MEMORY.md` ↔ 任务ID记录
- `./基础设定/TOOLS.md` ↔ 任务信息
- 各技能包 SKILL.md ↔ 实际配置

### 3. 自动修正
- 发现过期/已删除的定时任务 → 清理配置
- 发现信息不一致 → 同步更新
- 发现未记录的重要信息 → 补充记录
- 发现目录错放 → 移动并更新引用

### ⚠️ 文件操作安全规范（2026-04-27新增）

**严格执行以下规范，防止误删误改：**

| 场景 | 操作规范 |
|------|---------|
| 遇到重复文件 | **不轻易删除**，先比对内容差异、文件时间、大小，记录到报告等待人工确认 |
| 确需删除文件 | **移入回收站** `./回收站/YYYYMMDD/`，不直接删除（回收站保留7天） |
| 修改文件前 | **备份原文件**，命名：原文件名 + `_backup_YYYYMMDD` |
| 清理空目录 | 可直接移入回收站（风险较低） |

**禁止操作**：
- ❌ 直接删除任何文件
- ❌ 未比对就处理重复文件
- ❌ 未备份就修改重要配置

### 4. 经验总结
- 总结本周学到的经验教训
- 写入 TOOLS.md 供后续参考
- 提出优化建议

### 5. 聊天记录与项目文档检查（2026-04-30 新增）

**聊天记录检查：**
- 扫描 `./聊天记录/` 目录
- 识别聊天记录中未被沉淀到 `./知识沉淀/` 的高价值内容
- 提醒用户是否需要触发知识沉淀

**项目文档检查：**
- 扫描 `./项目文档/` 目录
- 检查所有标记「进行中」的项目是否超过24小时未更新
- 检查已完成项目是否遗漏了最终结论或待办未关闭
- 发现异常时提醒用户

## 执行流程

```
1. 目录结构检查
   - 扫描根目录是否有技能文件夹
   - 移动错放的目录
   - 搜索并更新所有路径引用

2. 读取当前所有定时任务配置
3. 对比各文件中的任务ID和信息
4. 扫描 ./聊天记录/ 目录
   - 读取近期的聊天记录文件
   - 识别未沉淀的高价值内容
5. 扫描 ./项目文档/ 目录
   - 检查「进行中」项目是否超24小时未更新
   - 检查已完成项目是否遗漏了最终结论
6. 自动修正不一致的地方
7. 生成复盘报告
8. 如有重要发现，通知用户
```

## 输出文件
- 复盘报告：`./复盘记录/YYYY-Wxx.md`
- 更新后的配置文件

## 检查清单

### 目录结构检查
- [ ] 根目录是否有技能文件夹需要移入 `./技能/`
- [ ] 移动后是否更新了所有路径引用
- [ ] 相关配置文件是否同步更新

### 配置同步检查
- [ ] tasks.json 任务列表完整
- [ ] MEMORY.md 任务ID正确
- [ ] TOOLS.md 信息同步
- [ ] 技能包 SKILL.md 配置正确

### 遗漏检查
- [ ] 新增技能是否记录
- [ ] 新增定时任务是否记录
- [ ] 修改的配置是否同步
- [ ] 重要的对话是否存档

### 成长检查
- [ ] 本周新增了什么能力
- [ ] 遇到了什么问题
- [ ] 学到了什么经验
- [ ] 有什么可以改进

## 反思与改进

### 问题根源分析
当发现问题时，要分析根本原因：
- 是流程问题？→ 优化工作流程
- 是规范问题？→ 补充规范文档
- 是习惯问题？→ 添加检查清单

### 持续优化
每次复盘都要问自己：
1. 有什么地方可以做得更好？
2. 有什么错误可以避免？
3. 有什么流程可以简化？
4. 有什么知识需要沉淀？
