# 文件清理技能 (file-cleaner)

## 技能信息
- 加载模式：progressive
- 触发关键词：[清理, 文件, 整理]
- 摘要：过期/临时文件自动清理

## 技能概述
智能文件清理技能，通过回收站机制安全管理垃圾文件，支持自定义清理规则、白名单保护、自动报告生成。

## 核心功能
1. **回收站机制** - 清理不直接删除，移入回收站保留7天，防止误删
2. **智能识别** - 自动识别临时文件、重复文件、过期缓存等垃圾文件
3. **白名单保护** - 重要文件永不清理，确保系统安全
4. **清理报告** - 每次清理生成详细报告，记录操作历史
5. **可配置规则** - 支持自定义清理规则和保留策略

## 使用方式

### 方式一：定时清理（推荐）
通过日程（Calendar）创建定时清理任务，按周期自动执行。

**创建日程示例：**
```
summary: 清理无用垃圾文件
dtstart: 202604260205 (凌晨2点05分)
rrule: 每2天执行一次
description: 使用 file-cleaner 技能执行文件清理
```

### 方式二：手动执行
通过 sessions_spawn 创建子任务，执行一次性清理：
```
任务描述：使用 file-cleaner 技能执行文件清理，清理当前工作目录
```

### 方式三：查看回收站
查看回收站内容、恢复文件或清空回收站：
```
任务描述：使用 file-cleaner 技能查看回收站状态
```

## 目录结构
```
./回收站/              # 回收站根目录
  └── YYYYMMDD/       # 按日期存放移入的文件

./清理记录/            # 清理报告目录
  └── 清理报告_YYYYMMDD.md

./技能/文件清理技能/  # 技能目录
  ├── SKILL.md         # 本文档
  ├── config.json      # 配置文件
  ├── references/      # 参考资料
  └── scripts/         # 辅助脚本
```

## 配置说明
编辑 `config.json` 可自定义以下内容：
- 清理规则：哪些文件类型被视为垃圾文件
- 白名单：永不清理的文件和目录
- 回收站保留天数（默认7天）
- 报告保存位置

## 注意事项
1. 重要文件已在白名单中保护，但仍需定期检查清理报告
2. 如需恢复回收站文件，请在7天内手动移出
3. 首次使用建议先手动执行一次，观察清理效果
4. 清理记录会长期保存，可定期整理
