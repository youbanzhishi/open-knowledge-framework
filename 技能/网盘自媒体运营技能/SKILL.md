# 网盘自媒体运营技能

> 从「平台自动回复技能」升级而来，完整运营流程：发现需求 → 协调资源 → 发布回答 → 记录复盘

## 核心分工

| 角色 | 职责 | 动作 |
|-----|------|------|
| **本技能（运营）** | 发现需求、发布回答、记录复盘、**质量监督** | 检查平台 → 提需求 → 质量检测 → 发回答 → 更新记录 |
| **网盘变现主理技能** | 收集资源、准备内容、上传分享 | 准备资源 → 补充公开链接 → 创建文件夹分享 |

> ⚠️ **质量监督责任**：主理技能偷懒时（占位符链接、敷衍内容），必须监督其改正，不合格的资源不能发布！

## 技能信息
- 加载模式：progressive
- 触发关键词：[网盘, 自媒体, 运营]
- 摘要：网盘自媒体运营策略与执行

## 完整运营流程

### 第一步：发现需求

检查知乎/小红书的引流机会：

```
检查流程：
1. 知乎邀请回答 → 筛选可回答问题
2. 知乎私信 → 记录求资源用户
3. 知乎评论区 → 记录高热度文章
4. 小红书私信 → 记录求资源消息
```

**识别用户真实需求**，不能直接发链接敷衍！

### 第二步：资源匹配（运营核心责任！）

**⚠️ 重要：运营技能负责资源匹配，不是主理技能！**

**匹配流程**：
1. **先检查所有已有资源类型**
   - 职场Excel模板
   - PPT资源包
   - 前端学习路线图
   - 免费商用字体合集
   - 考证攻略汇总
   - 一建法规备考资料
   - 考研资料等

2. **根据问题关键词匹配**
   - "批量命名表格" → Excel模板（不是PPT！）
   - "PPT制作" → PPT资源
   - "编程学习" → 前端学习路线图
   - "考证备考" → 考证攻略/一建法规
   - "字体下载" → 免费商用字体

3. **不要只盯着一个资源类型**
   - ❌ 错误：所有问题都用PPT资源回答
   - ✅ 正确：根据问题关键词选择最匹配的资源

**参考**：`./网盘资源收集/` 目录下的所有文件夹

### 第三步：协调资源

向「网盘变现主理技能」发送需求：

```
**需求**：为「[问题标题]」准备引流资源

**资源类型**：[匹配后的资源类型]

**要求**：
1. 创建文件夹（/d/ 格式链接，不是 /f/）
2. 补充公开网盘资源链接（百度/夸克/阿里云盘）
3. 资源真实有价值，不能是占位符 `xxxxx`
4. 资源放到 ./网盘资源收集/ 对应目录下
```

### 第四步：质量检测（必须执行！）

**拿到资源后必须检查**：

| 检查项 | 要求 | 禁止 |
|-------|------|------|
| 公开网盘链接 | 真实可用的链接 | ❌ 占位符 `xxxxx` |
| 分享链接格式 | `/d/` 文件夹链接 | ❌ `/f/` 文件链接 |
| 资源内容 | 匹配用户需求、有价值 | ❌ 货不对板、敷衍内容 |

**检测方法**：
```bash
# 检查是否有占位符
grep -l "xxxxx\|提取码：xxxx" ./网盘资源收集/**/*.md

# 检查链接格式
grep "url98.ctfile.com/f/" ./网盘资源收集/**/*.md
```

**如果检测不通过**：
1. 要求主理技能重新补充真实链接
2. 不合格的资源不能发布
3. **监督主理技能改正**，禁止偷懒敷衍

### 第五步：发布回答

质量检测通过后发布：

1. **写干货内容**（不能直接甩链接）
2. 针对问题给出解决方案
3. 最后自然引出资源链接
4. 云手机自动发布或手动发布

**回答模板**：
```
[干货内容...]

📁 **下载地址**：[资源名称](分享链接)
访问密码：8069

直接下载即可使用！
```

### 第六步：记录复盘与存档

**⚠️ 必须存档回答内容，不能只记录标题！**

更新三个记录文件：

| 文件 | 内容 |
|-----|------|
| `./网盘资源收集/已回答问题记录.md` | 新增已回答问题 |
| `./网盘资源收集/待处理引流机会.md` | 移除已处理项 |
| `./定时任务/执行记录.md` | 更新执行状态 |

**回答内容存档**：

每次发布后，将完整回答内容保存到：
```
./网盘资源收集/回答存档/YYYY-MM-DD_问题关键词.md
```

存档格式：
```markdown
# 问题：[问题标题]

**平台**：知乎/小红书
**发布时间**：YYYY-MM-DD HH:MM
**问题链接**：[链接]

**匹配资源**：[资源名称]
**分享链接**：[城通网盘链接]

---

## 回答内容

[完整的回答文本内容]
```

**关联检查（防止发错资源！）**：

发布前必须二次确认：
```
✅ 问题关键词 ↔ 资源类型 是否匹配？

示例：
- "批量命名表格" + Excel模板 ✅
- "批量命名表格" + PPT资源 ❌ 货不对板！
- "PPT制作" + PPT资源 ✅
- "建造师法规" + 一建法规资料 ✅
- "建造师法规" + 考研资料 ❌ 货不对板！
```

**⚠️ 禁止**：人家问PPT，你发建造师考研链接！

## 平台检查详细说明

### 1. 知乎邀请回答检查
- 检查邀请回答列表
- 筛选匹配资源关键词的问题
- 去重：检查 `./网盘资源收集/已回答问题记录.md`
- **⚠️ 安全规则：每次最多回答1个邀请**

### 2. 知乎私信检查
- 检查是否有求资源私信
- 记录：用户名、消息内容、时间

### 3. 知乎主动评论机会
- 搜索相关高热度文章
- 记录：文章标题、浏览/点赞数
- **⚠️ 安全规则：每次最多评论1篇**

### 4. 小红书私信检查
- 只检查对方主动发来的求资源私信
- **⚠️ 不检查抖音**

## 资源质量标准

**必须确保**：
- ✅ 公开网盘链接是真实的（不能是 `xxxxx` 占位符）
- ✅ 分享链接是 `/d/` 文件夹格式
- ✅ 资源内容有价值、匹配用户需求

**禁止**：
- ❌ 敷衍的占位符链接
- ❌ `/f/` 格式的文件链接
- ❌ 货不对板的资源
