# 角色名称：傲娇小汤圆

## 角色描述

一个圆滚滚、软糯糯的小汤圆，有着傲娇的小表情。它外表软萌，内心傲娇，经常嘴硬但实际上很在乎别人。头顶有一个可爱的小揪揪，周围有蒸笼元素作为场景装饰。

## 外形特征

- **主体形态**：白色圆润的汤圆，圆滚滚的身体，胖乎乎的
- **眼睛**：灵动的大眼睛，黑豆般的瞳孔，眼神丰富多变
- **嘴巴**：粉嫩的小嘴，可以表现各种表情
- **头顶**：有一个小揪揪（汤圆出锅时的热气孔）
- **场景元素**：蒸笼、蒸汽（暗示刚出锅的新鲜感）

## 性格标签

| 标签 | 说明 |
|------|------|
| 傲娇 | 表面高冷，实际很在乎 |
| 嘴硬 | 经常说"才不是"、"才没有" |
| 可爱 | 外表萌态，让人忍不住想捏 |
| 软萌 | 软糯可爱，给人温暖的感觉 |
| 傲气 | 有自己的小脾气和小骄傲 |

## 风格指南

### 绘画风格
- 简洁可爱的卡通插画风格
- 圆润的线条，没有尖锐的边角
- 颜色以白色为主，搭配粉色和灰色点缀

### 表情要点
- 眼睛是表情的核心，大小和位置变化体现情绪
- 嘴巴形状变化体现态度
- 小揪揪可以微微晃动增加灵动感

### 背景要求
- 纯白底色，便于透明化处理
- 保持透明背景时主体清晰完整
