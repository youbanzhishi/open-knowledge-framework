# 技能索引

> 知识体系三大支柱之一：技能目录
> 与项目/、角色/ 并列，共同构成知识体系骨架

## 目录结构
```
技能/
├── 索引.md                    ← 本文件
├── 打包备份/                  ← 定时备份任务
├── 知识体系同步/              ← Git推送任务
├── 文件清理/                  ← 垃圾清理任务
├── 网盘资源发布/               ← 网盘变现任务
├── 心跳检查/                  ← 守护监控任务
├── 自我复盘/                  ← 复盘任务
├── openvault-dev/             ← OpenVault项目开发
└── openlink-dev/              ← OpenLink项目开发
```
