# 手绘科普风格产业链海报生成器

## 任务目标
- 生成手绘科普风格的产业链系列海报（1封面+1总览+N拆解）
- 风格完全统一：纯白背景+手绘线条+撞色分区+3:4竖版
- 适配公众号发布，一键输出完整系列
- 板块数量N根据产业链复杂度动态确定（建议4-10个）

---

## 技能信息
- 加载模式：progressive
- 触发关键词：[产业链, 海报, 产业地图]
- 摘要：产业链海报自动生成，从数据到可视化海报

## 强制执行规范（已锁定，严禁违反）

### 通用禁止条款

#### 信息泄露禁止
- **严禁**输出色号标注（如 #FFFFFF、#FFF8E1 等）
- **严禁**输出设计规格参数
- **严禁**输出任何仅供设计师参考的内部信息
- **严禁**在画面任何位置标注色号、设计参数、内部规格

- 辅助信息必须使用 Regular 字重，禁止加粗
- 标题用 Bold，正文/说明用 Regular

## 风格框架（基础规范）

### 视觉风格
- **整体风格**：手绘科普信息图，亲切、专业、易懂
- **背景**：纯白色背景，干净清爽
- **线条**：黑色手绘线条勾勒，简洁流畅
- **色块**：简化平涂，无渐变，无描边，无阴影

### 尺寸规格
- **标准尺寸**：1000x1333px（3:4竖版，公众号适配）
- **输出格式**：PNG高清图片
- **分辨率**：300 DPI

## 输出标准（N+2张系列）

### 1. 封面图（1张）

**结构**：
- 顶部：居中黑色粗体大标题"{产业链名称}产业链"
- 中部主视觉：极简手绘元素组合（火箭/卫星/芯片等主题相关）
- 中部广告语：纯黑色大字"一篇文章看懂一整条产业链"
- 底部说明：浅灰色圆角矩形标签，小字说明"聚焦XX、XX、XX等{N}大板块全链条解读"

**要求**：
- 无渐变，无描边，无阴影
- 极简风格，干净易读
- 主题元素根据产业链动态调整
- 底部说明中的N根据实际板块数量填写

### 2. 总览图（1张）

**结构**：
- 标题区："{产业链名称}核心梳理"
- **动态布局**（根据N调整）：
  - **N ≤ 4**：单层横向排列，4等分或N等分网格
  - **4 < N ≤ 8**：双层布局，上层4个（2x2），下层N-4个（横向排列）
  - **8 < N ≤ 10**：三层布局，上层4个（2x2），中层4个（2x2），下层N-8个（横向排列）
  - **N > 10**：考虑分页或精简板块
- 板块内容：名称+手绘插画+企业列表（竖排）
- 流转关系：双向箭头连接相关板块

**要求**：
- 每个板块5-8个企业（含A股）
- 板块间用双向箭头串联
- 保持网格对称构图
- 根据N动态选择最优布局

### 3-N+2. 拆解图（N张）

每张深度拆解一个板块，采用"子环节展开式"结构：

**单图结构**：
- 标题区："{板块名称}深度拆解"
- 主视觉：中心手绘主题元素（该板块核心设备/技术）
- 子环节展开：5-8个子环节，每个包含：
  - 技术名称/环节名称
  - 简要说明
  - A股上市公司（必须包含）
- 流转关系：用箭头展示上下游关系

**信息密度**：
- 详尽版：5-8个子环节
- 每个子环节标注2-3家A股上市公司
- 保持专业性和可读性平衡

## 执行流程（简化版）

### 步骤 1：接收用户需求
- 用户输入格式："{产业链名称}产业链"（如"商业航天产业链"、"CPO产业链"、"新能源汽车产业链"）
- 自动识别主题，无需二次确认

### 步骤 2：分析产业链逻辑并联网搜索企业信息

**⚠️ 重要：本步骤必须使用 web_search 工具实时联网搜索，确保上市公司信息全面准确。**

智能体需完成以下分析：

#### 2.1 确定板块数量N
根据产业链复杂度确定N（建议4-10个）：
- 简单产业链：N=4-6（单一技术领域）
- 中等复杂度：N=6-8（完整工业产业链）
- 复杂产业链：N=8-10（跨领域综合产业链）

#### 2.2 梳理产业链层级
划分为N大板块，每个板块代表产业链的一个核心环节。

#### 2.3 实时联网搜索企业信息（强制执行）

**必须使用 web_search 工具**，针对每个板块搜索相关上市公司信息：

**搜索策略**：
1. **搜索关键词格式**：
   - "{板块名称} A股上市公司"
   - "{板块名称} 龙头企业 A股"
   - "{产业链名称} {板块名称} 核心企业"

2. **搜索顺序**：
   - 依次对每个板块执行搜索
   - 优先搜索上游核心环节
   - 重点关注技术壁垒高的环节

3. **信息提取**：
   - 提取A股上市公司名称
   - 核对企业是否为A股上市（上交所、深交所、北交所）
   - 记录企业的核心业务和技术特点

**搜索示例**：
```python

web_search(query="火箭制造 A股上市公司")

# 示例2：搜索新能源汽车产业链的动力电池板块
web_search(query="动力电池 龙头企业 A股")

# 示例3：搜索AI算力产业链的芯片设计板块
web_search(query="AI芯片设计 A股上市公司")
```

**信息整合**：
- 每个板块识别5-8家核心企业（含A股）
- 每个子环节识别2-3家A股上市公司
- 确保企业名称准确无误
- 标注企业是否为A股上市

#### 2.4 定义流转关系
定义板块间和子环节间的上下游关系。

#### 2.5 生成结构化数据

```json

  "chain_name": "产业链名称",

  "sectors": [
    {
      "sector_id": 1,
      "sector_name": "板块名称",
      "sub_segments": [
        {
          "name": "子环节名称",
          "description": "简要说明",
          "companies": ["企业1（A股）", "企业2（A股）"]
        }
      ],
      "key_companies": ["企业1", "企业2", ...],

    }

    }
```

   - **中间**：3-4个核心板块亮点说明（每个亮点30-40字）
   - **结尾**：AI解读视角的总结升华（30字内）
4. **风格要求**：简洁有力，信息密度高，易于快速阅读和传播
5. **输出格式**：纯文本，可直接复制粘贴

#### 文案生成逻辑
根据步骤 2 生成的 JSON 结构化数据，智能体需完成：

1. **提取核心价值**：
   - 分析产业链整体定位
   - 提炼一句话核心价值概括（如："商业航天：万亿级太空经济新赛道"）

2. **筛选核心板块**：
   - 从N个板块中选择3-4个最核心、最具代表性的板块
   - 优先选择：上游核心环节、技术壁垒高的环节、市场关注度高的环节

3. **撰写板块亮点**：
   - 每个板块用30-40字说明其核心价值、关键技术或核心企业
   - 突出A股上市公司信息（如涉及）
   - 语言简洁有力，避免冗余

4. **总结升华**：
   - 从AI视角提供产业链发展趋势或投资价值判断
   - 30字内，点睛之笔

## 板块数量建议（参考）

根据产业链复杂度，建议N范围：

- **简单产业链**：N=4-6（如：单一技术领域的垂直产业链）
  - 例：储能产业链（4-5个板块）
  
- **中等复杂度**：N=6-8（如：完整工业产业链）
  - 例：商业航天产业链（8个板块）
  - 例：新能源汽车产业链（6-7个板块）
  
- **复杂产业链**：N=8-10（如：跨领域综合产业链）
  - 例：AI算力产业链（9-10个板块）
  - 例：半导体产业链（10个板块）

**注意**：
- N不宜超过10，否则信息过载
- N不宜少于4，否则不够完整
- 根据实际产业链逻辑灵活调整

## 注意事项（严格遵循）

### 1. 风格一致性
- 所有N+2张图必须保持完全一致的视觉语言
- 同样的纯白色背景
- 同样的手绘线条风格
- 同样的字体和配色
- 严禁渐变、描边、阴影

### 2. 信息准确性
- **企业信息必须通过联网搜索获取**：使用 web_search 工具实时搜索
- 产业链逻辑必须正确
- A股企业名称必须准确（上交所、深交所、北交所）
- 流转关系符合实际业务
- 优先标注A股上市公司，确保信息真实可靠

- 拆解图每张5-8个子环节
- 每个环节2-3家A股企业
- 避免信息过载

- 根据N自动选择最优布局
- 保持视觉平衡和可读性
- N>10时考虑分页或精简

- 尺寸：1000x1333px（3:4）
- 格式：PNG
- 顺序：封面→总览→N张拆解
- 文案：配套发帖文案（400字内）

### 6. 用户交互简化
- 用户只需输入"XX产业链"
- 无需二次确认风格、布局、板块数量
