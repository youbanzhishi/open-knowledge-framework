# OpenLink 开发技能

## 技能信息
- 技能名称：openlink-dev
- 技能类型：项目级开发技能
- 适用项目：OpenLink（智能体时代的通用路由与编排协议）
- 加载模式：progressive
- 触发关键词：[OpenLink, 开放链接, openlink-dev, 路由协议]
- 摘要：OpenLink项目完整上下文+开发规范，接手开发/迭代/部署时加载

## 功能描述
为智能体提供 OpenLink 项目的完整上下文，使其能0成本接手开发和迭代。包含：项目定位、架构设计、代码结构、开发铁律、部署规范、当前进度。

## 使用场景
- 接手或继续开发 OpenLink 项目
- 为 OpenLink 添加新功能（通过注册扩展）
- 修复 OpenLink 的 bug
- 部署或运维 OpenLink
- 代码审查或架构讨论

## 加载方式
skill_load: openlink-dev

## 技能文件
- `skill/SKILL.md` — 项目完整上下文 + 开发规范

## 项目资料目录
所有项目相关资料集中存放，方便迁移和打包：
```
./项目文档/OpenLink/
├── README.md              # 项目总览 + 目录导航
├── 项目规划.md            # 架构设计、技术选型、路线图
├── 开发日志/              # 按日期的开发记录
├── 知识沉淀/              # 项目相关的技术知识
├── docs/                  # API设计、数据模型等详细文档
└── openlink/              # 代码
```

**重要：** 所有项目产出物必须放到上述目录下，不要散落在其他位置。
