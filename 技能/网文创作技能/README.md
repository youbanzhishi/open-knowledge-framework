# 网文创作技能

起点中文网男频玄幻小说半自动创作全流程辅助。

## 目录结构

```
网文创作技能/
├── SKILL.md                          # 主文件（能力说明+工作流程+写作规范+自检流程+修改规范）
├── README.md                         # 本文件
└── references/
    ├── 写作经验案例库.md              # 14个真实改前改后案例
    ├── 自检扫描模板.md               # 7维度扫描检查项+输出格式
    ├── 章节细纲模板.md               # 细纲标准格式
    ├── 起点发布规范.md               # 平台政策+半自动模式+发布流程
    └── 去AI味检查清单.md             # 禁止词表+禁止句式+高频问题速查
```

## 使用方式

1. **加载技能**：`skill_load` → `网文创作技能`
2. **创建任务**：通过 `sessions_spawn` 启动sub-agent，任务描述中注明"使用网文创作技能"
3. **技能主文件**：SKILL.md 包含完整工作流程和写作规范
4. **参考文件**：references/ 下有详细模板和案例库

## 核心特点

- **半自动模式**：AI负责大纲/细纲/初稿/自检/润色，用户主导正文修改，符合起点平台政策
- **自检驱动**：每章初稿后必须7维度自检，给用户A/B/C选项，用户确认后才改
- **经验沉淀**：14个真实案例覆盖爆发点、画面连贯性、对话语气、去AI味等核心经验
- **可迁移**：不绑定具体小说项目，核心规范适用于所有起点男频玄幻小说
