# 数据统计技能

## 技能信息
- 加载模式：progressive
- 触发关键词：[统计, 数据, 报表]
- 摘要：数据统计与报表生成

## 技能概述
网盘变现项目的数据分析系统，实现各平台数据统计、趋势分析、策略调整建议。

## 功能特点
- 📊 各平台文章数据统计（阅读、点赞、评论等）
- 📥 城通网盘下载数据统计
- 📈 趋势分析与热点发现
- 🔄 自动策略调整建议
- 📝 周报自动生成

## 使用场景
- 每周数据统计分析
- 发现热门资源类型
- 评估平台表现
- 生成运营周报

## 执行流程

### 完整统计流程
1. **收集各平台数据** → 云手机App查看文章数据
2. **收集网盘下载数据** → 城通网盘后台或API
3. **汇总分析** → 计算各项指标
4. **生成周报** → 保存到指定目录
5. **策略调整** → 根据规则提出建议

### 数据采集方式

#### 各平台数据
| 平台 | 数据项 | 采集方式 |
|------|--------|---------|
| 知乎 | 浏览量、点赞、评论 | 云手机App |
| 微博 | 阅读量、转发、评论、点赞 | 云手机App |
| 小红书 | 浏览量、收藏、评论、点赞 | 云手机App |
| CSDN | 阅读量、评论 | 网页后台 |
| 博客园 | 阅读量、评论 | 网页后台 |

#### 网盘数据
- 各文件夹下载次数
- 各文件下载次数
- 总下载量统计

## 策略调整规则

### 自动决策规则
| 条件 | 操作 | 优先级 |
|------|------|--------|
| 下载量 > 平均值2倍 | 该类资源优先级提高 | 高 |
| 下载量 < 平均值50% | 该类资源暂停/减少 | 中 |
| 单篇互动 > 100 | 考虑发布续篇或系列 | 中 |
| 某平台表现突出 | 加大该平台投入 | 高 |
| 某平台持续低迷 | 减少该平台发布 | 低 |

### 通知规则
- 🔴 重大变化：立即通知用户
- 🟡 中等变化：写入周报
- 🟢 轻微波动：仅记录

## 调用示例

### 每周统计任务
```
加载数据统计技能，执行本周数据分析。
```

### 查看特定平台
```
加载数据统计技能，分析知乎平台近期表现。
```

### 紧急数据分析
```
加载数据统计技能，分析最近3天的数据变化。
```

## 输出文件
- 周报：`./网盘引流项目/周报/{YYYY}-W{xx}.md`
- 数据记录：`./网盘引流项目/周报/data_{YYYY}-W{xx}.json`
- 调整建议：发送消息通知用户

## 注意事项
1. 数据采集需要登录各平台
2. 统计周期为自然周（周一到周日）
3. 重大变化需要及时通知用户
4. 周报保存后同时更新MEMORY.md
