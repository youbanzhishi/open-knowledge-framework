# 拖鞋大冒险

> 荒诞武器驱动的2.5D纵深横版闯关游戏

## 项目定位

**拖鞋大冒险**——用荒诞打破认知惯性，让每关都有"卧槽"瞬间。

- 类型：2.5D纵深横版闯关（Beat-em-up）
- 引擎：Cocos Creator 3.x + TypeScript
- 目标平台：微信小游戏/H5 → PC/手游

## 快速上手

### 标志性场景
蚁群围攻 → 爬树 → 抱核弹头飞跃砸地 → 全灭

### 核心玩法
- **9种荒诞武器**：拖鞋、吐口水、放屁、粑粑投弹、喷嚏冲击、舌头鞭打、呕吐喷射、内裤套头、蚁群核弹
- **铁锅防御**：举锅格挡 + 松开扣锅反击
- **触屏操控**：虚拟摇杆 + 按键
