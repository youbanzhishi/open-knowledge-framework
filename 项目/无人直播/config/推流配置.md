## 抖音推流码获取（待完成）

### 官方获取方式

抖音推流码**必须通过官方客户端实时获取**，每次开播都会生成新的推流地址。

#### 方式1：抖音直播伴侣（PC端）

1. 下载并安装 [抖音直播伴侣](https://www.douyin.com/live)
2. 登录账号（需实名认证）
3. 点击「开始直播」
4. 选择直播类型（游戏/娱乐/教学等）
5. 在直播设置界面，开启"显示推流地址"选项
6. 复制 RTMP 地址和推流密钥

#### 方式2：手机App（需1000+粉丝）

路径：抖音APP → 开直播 → 选择"电脑直播" → 查看推流地址

### 推流地址格式

```
RTMP服务器: rtmp://push.douyin.com/live/
推流密钥: [动态生成的密钥字符串]
```

完整推流地址示例：
```
rtmp://push.douyin.com/live/xxxxx?streamKey=xxxxx
```

### ⚠️ 重要说明

- 推流码**仅在当次直播有效**，关闭直播后立即失效
- 每次开播需要重新获取推流地址
- 普通账号可能需要1000粉丝才能开启推流权限
- 企业认证账号可获得更稳定的推流能力

## 测试推流命令示例

### 抖音推流（需替换实际推流码）

```bash
ffmpeg -re -stream_loop -1 -i video.mp4 \
  -c:v libx264 -preset ultrafast -b:v 2000k \
  -c:a aac -b:a 128k \
  -f flv "rtmp://push.douyin.com/live/xxxxx?streamKey=xxxxx"
```

### FFmpeg循环推流参数

| 参数 | 说明 |
|------|------|
| `-re` | 按原始帧率读取（实时推流） |
| `-stream_loop -1` | 无限循环视频 |
| `-preset ultrafast` | 低延迟编码 |
| `-c:v libx264` | H.264视频编码 |
| `-c:a aac` | AAC音频编码 |
