# PhoneAgent - 自建手机远程控制工具

> 最后更新：2026-05-11
> 轻量运营项目 — 云手机远程控制，服务端+客户端+部署

## 项目概述
远程控制云手机的工具，支持ADB设备管理、视觉识别、任务调度、自动重连。

## 源码仓库
- **GitHub**: https://github.com/youbanzhishi/phone-agent （私有）
- 服务端：Flask API + ADB + 视觉识别
- 客户端：ADB代理 + 自动重连
- 部署：Docker + deploy.sh 一键部署
