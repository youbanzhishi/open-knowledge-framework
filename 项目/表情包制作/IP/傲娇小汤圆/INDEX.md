## 本地偏差段

| 偏差 | 说明 |
|------|------|
| 参考图按性别区分 | 雄性用不带文字版，雌性用带文字版 |
| 原图用绿幕/红幕 | 方便抠透明图，衍生图才用白底 |
| 新年版用红底 | 用户偏好红底喜庆风，虽然头顶不如白底精准但整体效果更佳 |
| 运动版参考图更稳 | 新年版v3测试发现运动版精选图做参考图头顶最准确，但用户最终选择旅游版参考图+红底 |
