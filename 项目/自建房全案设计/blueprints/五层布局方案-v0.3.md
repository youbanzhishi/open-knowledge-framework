## 一、"堡垒"设计落地（新增核心）

### 设计理念来源
> 业主小时候喜欢用被子把自己裹住——"感觉在一个无坚不摧的堡垒中，能看到外界一切，外界却察觉不到我"

这不是装修风格，是**空间心理学设计**，贯穿每一层。

### 堡垒五要素落地

#### 要素1：外界看不到我（私密性）

| 空间 | 做法 | 效果 |
|------|------|------|
| 2F客厅 | 朝马路方向：低透LOW-E玻璃（透光不透影） | 白天外面看不清里面 |
| 2F空中明堂 | 外侧装磨砂玻璃栏板 | 在明堂活动不被对面看到 |
| 3F儿童房 | 朝马路窗：内嵌百叶帘 | 随时可调视角 |
| 4F练舞房 | 朝马路窗：中空隔音玻璃 | 练习不被窥视 |
| 所有窗户 | 标配内遮光帘 | 晚上室内活动不透光 |

#### 要素2：我能看到外界（观察孔道）

| 空间 | 做法 | 效果 |
|------|------|------|
| 1F车库入口 | 高位观察窗（离地1.8m，面积30×20cm） | 门外有人我不暴露位置 |
| 1F安全厅 | 猫眼+摄像头双通道 | 不开门知门外情况 |
| 3F男孩/女孩房 | 高位细长横窗（离地1.6m，宽60cm高20cm） | 躺床上也能看窗外，不露脸 |
| 4F练舞房 | 整面墙镜子朝室内 | 外面看不到里面，里面看外面 |
| 屋顶花园 | 朝马路方向设瞭望台（0.5㎡） | 观察全村，不被看到 |

#### 要素3：无坚不摧的结构感

| 空间 | 做法 | 效果 |
|------|------|------|
| 整体外观 | 城堡式斜坡屋顶+外墙砖砌感 | 厚重、防御性强 |
| 1F外墙 | 比标准墙厚10cm（37墙） | 结构感+隔音+隔热 |
| 大门 | 装甲门（防火防盗）+钢制门框 | 堡垒感第一关 |
| 1F安全厅 | 加固墙体+钢板门+监控屏 | 紧急情况最后防线 |
| 楼梯间 | 钢筋混凝土整体浇筑 | 火灾时唯一安全通道 |

#### 要素4：领地感（我有我的空间）

| 空间 | 做法 | 效果 |
|------|------|------|
| 儿童房内 | 每孩独立衣柜+书桌+床 | 明确"这是我的" |
| 主卧 | 独立卫浴+衣帽间+书房角 | 夫妻私人领地 |
| 4F创作区 | 每个功能独立房间（录音/练舞/影院分离） | 工作室各自独立 |
| 屋顶花园 | 分隔种植区+鸡舍 | 各有"自留地" |

#### 要素5：内向聚拢的空间组织

| 空间 | 做法 | 效果 |
|------|------|------|
| 整体布局 | 中心天井+外围房间 | 气聚不散 |
| 2F空中明堂 | 半封闭阳台，花池围合 | 私密庭院感 |
| 屋顶花园 | 四周女儿墙围合 | 私属空中领地 |
| 声学空间 | 录音室/影院全封闭 | 声音不外泄 |

## 二、儿童分房扩展方案（核心新增）

### 家庭生命周期预测

```
现在(2026)：5孩 → 男孩房(2人) + 女孩房(3人)
     ↓
5年后(2031)：老大~20岁，可能外迁/住校，4孩
     ↓
10年后(2036)：5孩均18+，各自独立需求强烈
     ↓
娶媳妇(2040+)：媳妇随夫居，需要更多独立空间
     ↓
老人离世(不确定)：老人房可释放，但届时已有更多人口
```

### v0.3 预留设计

#### 预留原则：结构不动，管道和空间先行

| 预留项 | 现在做什么 | 未来怎么用 |
|--------|----------|-----------|
| 墙体预留 | 3F儿童房隔墙用可拆式轻钢龙骨墙 | 拆墙=1间变2间 |
| 电线管道 | 每孩床位处预埋独立电路+网络线 | 分房时不断电不断网 |
| 管道井 | 3F预留横向管道井 | 分房后各自卫浴排水 |
| 门洞位置 | 轻钢龙骨墙预开门洞位置 | 未来开门即独立房间 |
| 结构柱 | 3F层高2.8m，留足天花板空间 | 未来搭阁楼夹层 |

#### 3F分房扩展方案图

```

┌─────────────────┐           ┌────┬────┬────┐
│                 │ 拆两道     │    │    │    │

```

#### 分房扩展操作指南（写入项目文档）

| 阶段 | 时间 | 操作 | 成本 |
|------|------|------|------|
| 一期建 | 2026 | 建轻钢龙骨隔墙（可拆） | 含在主体造价内 |
| 二期改 | ~2036 | 拆隔墙+开门洞+安新门 | ~5000元/间 |

## 四、各层详细布局（v0.3 更新）

### 1F — 基础层（含堡垒设计）

```
┌─────────────────────────────────────────────┐
│  前方(东北66°) → 大门                        │
├───────────┬──────────┬──────────────────────┤
│ 车库(30㎡)│ 楼梯间   │ 观察窗               │
│           │(8㎡)    │ (高位猫眼式)          │
│           │          ├──────────────────────┤
│           │          │ 修理间(15㎡)          │
├───────────┴──────────┤                      │
│                       ├──────────────────────┤
│ 老人房(18㎡)         │ 算力机房(15㎡)         │
│ (西南·天医方)         │ (暗区·热能回收)       │

├─────────────────────┼──────────────────────┤
```

**1F堡垒要素：**
- 观察窗（高位30×20cm）→ 门外动静我知道，我不暴露
- 安全厅（钢板门+监控）→ 最后防线
- 37墙 → 结构感+隔音
- 老人房内帘 → 私密感
