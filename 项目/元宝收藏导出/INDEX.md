# 元宝收藏导出

> 从腾讯元宝 App/网页版导出收藏文章，按指定格式整理为 Markdown 文件，支持自动分类和定时任务

> ⚠️ **隐私声明**：本项目包含私人博客链接和签名模板，仅限个人使用

## 执行步骤

### 完整导出流程

1. **初始化**：读取索引文件，加载已导出标题列表
2. **登录认证**：使用 mobile_use 打开元宝App → 手机号18630620063 + 验证码登录
3. **获取收藏列表**：进入收藏页面，向下滚动加载更多
4. **导出前去重**：对比索引，跳过已导出，仅处理新文章
5. **提取内容**：标题、收藏时间、正文（不含"用户提问"和"AI回复"标题）
6. **智能分类**：根据内容关键词自动判断分类
7. **格式生成**：统一 Front Matter + 正文 + 版权声明
8. **保存归档**：保存到对应分类目录，更新索引文件

### 批量导出

- 每批次10篇
- 批次间保存进度
- 遇到错误可断点续传

## 注意事项

### 执行原则

- 导出后必须对比截图确认内容完整，不能只看文件生成就认为完成
- 优先使用元宝App复制功能而非截图
- 格式修改后需自检修复
- 发现问题必须立即修正
- 执行时必须核对App名称，避免打开错误应用
- 删除旧文件必须移至回收站

### 格式规范

- 文件命名：不带序号前缀（如"Python异步编程实战"而非"01_Python异步编程实战"）
- Front Matter：title + date（精确到秒）+ tags（单个核心关键词，无中括号）
- 正文直接开始，不加"用户提问"和"AI回复"标题
- 正文后空三行再写版权声明

### 常见问题

- 导出内容不完整（只有692B）→ 排查是否只导出了部分内容
- AI对话痕迹未清理干净 → 检查并移除
- 格式修改后仍有问题 → 检查是否有多余空行或格式标记
