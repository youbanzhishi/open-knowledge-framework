## 分类规则

### IT技术类
**关键词：**
- Docker, docker-compose, 容器
- Python, Java, JavaScript, 编程
- Linux, 服务器, 部署
- API, OpenClaw, 配置
- Git, GitHub, 版本控制
- MySQL, 数据库, Redis
- Nginx, 网关, 反向代理

**标签提取：**
- Docker 相关 → `Docker`
- OpenClaw 相关 → `OpenClaw`
- 数据库相关 → `MySQL` / `Redis`
- 编程语言 → 对应语言名

### 中医养生类
**关键词：**
- 气功, 站桩, 静坐
- 经络, 穴位, 通经络
- 气血, 补气血, 养气血
- 养生, 中医, 功法
- 肝郁, 脾虚, 脏腑
- 内功, 真气, 呼吸

**标签提取：**
- 气功/站桩相关 → `气功`
- 经络相关 → `经络`
- 气血相关 → `气血`
- 功法相关 → `功法`

### 金融类
**关键词：**
- ETF, 基金, LOF
- 股票, 投资, 理财
- 融资融券, 杠杆
- 网格策略, 定投
- 猪周期, 行业周期
- 百万医疗, 防癌险

**标签提取：**
- ETF 相关 → `ETF`
- 股票投资 → `投资`
- 保险相关 → `保险`

### 音乐音频类
**关键词：**
- 编曲, 作曲, 乐理
- 混音, 母带, 后期
- 音频, 声卡, MIDI
- DAW, 宿主, 插件
- 合成器, 采样, 音色

**标签提取：**
- 编曲相关 → `编曲`
- 混音相关 → `混音`
- 音频技术 → `音频`

### 动漫游戏建模绘画类
**关键词：**
- 动漫, 动画, 二次元
- 游戏, 游戏开发, 游戏设计
- 建模, 3D, Blender, Maya
- 绘画, 插画, 原画
- 角色设计, 场景设计

**标签提取：**
- 建模相关 → `建模`
- 绘画相关 → `绘画`
- 游戏相关 → `游戏`
