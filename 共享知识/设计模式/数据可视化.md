# 数据可视化技能（ECharts）知识提炼

> 来源：./技能/数据可视化技能/SKILL.md
> 提炼时间：2026-05-09

## 踩坑经验

| 坑 | 症状 | 解法 | 严重度 |
|----|------|------|--------|
| ECharts配置有函数 | 图表渲染失败 | 用模板占位符如`{b}: {c}`替代 | 🔴渲染失败 |
| 词云用随机颜色函数 | 渲染失败 | 每个data项中指定`textStyle.color` | 🔴渲染失败 |
| 尾逗号 | JSON解析失败 | 移除所有尾逗号 | 🔴语法错误 |
| 单引号 | JSON解析失败 | 使用双引号 | 🔴语法错误 |
| 注释在JSON中 | 解析失败 | 移除所有`//`和`/* */`注释 | 🔴语法错误 |
| 变量引用 | 渲染失败 | 直接写数值/字符串，不用变量 | 🔴语法错误 |
| tooltip中获取颜色 | 渲染失败 | 用固定颜色或series.itemStyle | 🔴功能限制 |
| 多grid布局 | 布局混乱 | 只用一个grid | 🟠布局问题 |

## 最佳实践

### 输出规范
- **输出纯JSON**：禁止函数、注释、尾逗号
- **直接输出代码块**：不在前面加"XX配置"等标题
- **禁止作为单独文件输出**：echarts json直接内嵌

### 图表类型选择
| 场景 | 推荐 | 备注 |
|------|------|------|
| 时间序列趋势 | 折线图 | - |
| 分类比较 | 柱状图 | 类别≤6纵向，>6横向 |
| 占比≤5类 | 饼图 | 独占series |
| 占比>5类 | 排序柱状图 | 降序展示 |
| 文本频次 | 词云 | - |

### 多指标对比选择
| 情况 | 选择 |
|------|------|
| 同单位、量级相近 | 多线图 |
| 量 + 率 | 柱状 + 折线 |
| 量级差异大 | 拆成两图（慎用双Y轴） |

### 颜色设置
- **series层级设置**：legend自动继承
- **饼图/词云**：每个data项单独指定
- **正负值区分**：data项中指定`itemStyle.color`

### 标题撰写
- 应传达洞察而非仅描述数据
- 避免："月度销售数据"
- 推荐："Q3销售额增长40%"

### 布局规范
```json
{ "grid": { "left": "6%", "right": "6%", "bottom": "3%", "containLabel": true } }
```

### 格式化模板
| 占位符 | 含义 |
|--------|------|
| `{a}` | 系列名 |
| `{b}` | 类别 |
| `{c}` | 值 |
| `{d}` | 百分比（仅饼图）|

### 数据预处理
- 长标签缩写："2024财年..." → "Q3华东"
- 大数值换算单位：数据用`[10, 20]`而非`[100000, 200000]`
- 数值保持为number类型，不转字符串

## 平台规则/限制

### 默认色板
- **温暖简约**：`["#d97757", "#6a9bcc", "#788c5d", "#c4a35a", "#8b7cb6", "#c97b84", "#5d8c8c", "#a67c52", "#b0aea5"]`
- **商务风格**：`["#5470c6", "#91cc75", "#fac858", "#ee6666", "#73c0de", "#3ba272", "#fc8452", "#9a60b4", "#ea7ccc"]`

### 禁用项
- ❌ 雷达图、甘特图、热力图（除非用户明确要求）
- ❌ 函数作为配置值
- ❌ 注释
- ❌ 尾逗号
- ❌ 单引号
- ❌ 变量引用
- ❌ 多grid布局
- ❌ tooltip.formatter中获取颜色

## 设计决策

| 决策 | 选项 | 最终选择 | 理由 |
|------|------|----------|------|
| JSON格式 | 允许函数 | 禁止函数 | 模板占位符足够用 |
| 颜色来源 | 随机/固定 | 固定色板 | 风格统一+可预期 |
| 标签显示 | 全显示/按需 | 数据点≤6直接显示 | 避免遮挡 |
| 双Y轴 | 允许/限制 | 需scale:true | 避免数据被压平 |

## 技术架构

### ECharts核心配置
```json
{
  "title": { "text": "洞察标题" },
  "legend": { "top": "top" },
  "grid": { "left": "6%", "right": "6%", "bottom": "3%", "containLabel": true },
  "xAxis": { "type": "category", "data": [...] },
  "yAxis": { "type": "value" },
  "series": [{
    "type": "bar",
    "name": "系列名",
    "itemStyle": { "color": "#d97757" },
    "data": [...]
  }]
}
```

## 可升级为热规则的警告

- ⚠️ **JSON中禁止任何函数**：包括formatter、color函数等，渲染必失败
- ⚠️ **词云颜色必须在data项中指定**：不能用函数随机
- ⚠️ **禁止尾逗号/单引号/注释**：JSON语法严格
- ⚠️ **数据预处理前置**：长标签缩写、大数值换算后再输出
- ⚠️ **饼图独占series**：不能与其他类型混合
