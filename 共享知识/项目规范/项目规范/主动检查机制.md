# 主动检查机制

> 用于自动化协作循环：每次AI响应前检查任务队列，有任务立刻处理

## 触发条件

AI每次响应消息前，先检查 `/tmp/任务队列.md`

## 检查逻辑

```bash
if [ -f /tmp/任务队列.md ]; then
    # 读取任务
    # 提取角色和工单
    # 切换角色处理
    # 完成后更新工单状态
    # 删除任务队列
fi
```

## 任务队列格式

```markdown
## 待处理任务队列

### 发现时间
2026-05-19 10:00:00

### 工单列表
- **[角色名]** WO-xxx 工单名称 (优先级: P1)
- **[角色名]** WO-xxx 工单名称 (优先级: P2)
```

## 处理流程

1. 读取队列中的第一个工单
2. 切换到对应角色
3. 按五步门执行
4. 更新工单状态为已完成/处理中
5. 再次检查队列
6. 循环直到队列为空

## 配置方式

在AI的系统提示词或启动脚本中加入检查逻辑
