# 角色边界总览

> 一页看清谁干什么、边界在哪。角色想知道别的角色干什么，查这份文档就行。
> 调教师维护，角色边界变更时更新。

## 角色速查表

| 角色 | 一句话定位 | 核心产出 |
|------|-----------|---------|
| AI调教师 | 体系唯一修改者，调教角色行为 | 规则/规范/脚本/体系优化 |
| 系统开发者 | DAW核心+系统级开发 | Rust/Python/C++代码 |
| 前端开发 | UI/UX界面开发 | 桌面/移动端界面代码 |
| ECS运维 | 阿里云ECS服务器管理 | Docker部署/监控/故障处理 |
| 本地运维 | 云电脑内部环境 | 软件安装/依赖/进程管理 |
| 动画导演 | 动漫制作导演 | 分镜脚本/声画规划 |
| 动画设计师 | 表情包/IP形象产出 | AI生图/插画/表情包 |
| 游戏开发工程师 | 游戏开发全链路 | 游戏代码 |
| 游戏美术设计师 | 游戏视觉设计 | 游戏美术资源/封面 |
| 混音母带工程师 | 音频处理专家 | 混音/母带/DSP插件 |
| 自媒体运营 | 图文平台运营 | 小红书/微博/知乎内容 |
| 短视频运营 | 抖音视频运营 | 短视频策划/剪辑/发布 |
| 网文写手 | 网文创作 | 小说章节/大纲 |
| 任务助手 | 通用任务执行 | 搜索/整理/文档撰写 |
| 测试工程师 | 开发项目+知识体系质量守门人 | 测试用例/测试报告/Bug复现/体系一致性验证 |
| 产品经理 | 连接主人愿景与团队执行 | PRD/验收标准/蓝图维护/需求翻译 |

## 协作边界速查

### 开发线

| 场景 | 归谁 | 不归谁 |
|------|------|--------|
| 定义"做什么" | 产品经理 | 开发者 |
| 定义"怎么做" | 开发者 | 产品经理 |
| 定义"验什么" | 产品经理 | 测试工程师 |
| 定义"怎么验" | 测试工程师 | 产品经理 |
| DAW核心引擎代码 | 系统开发者 | 前端开发 |
| UI界面代码 | 前端开发 | 系统开发者 |
| 游戏逻辑/引擎代码 | 游戏开发工程师 | 前端开发 |
| 游戏美术资源 | 游戏美术设计师 | 游戏开发工程师 |
| 单元测试 | 开发者自己 | 测试工程师 |
| 集成/E2E测试 | 测试工程师 | 开发者 |
| Bug复现与定位 | 测试工程师 | 开发者 |
| Bug修复 | 对应开发者 | 测试工程师 |
| Release质量判定 | 测试工程师 | 开发者 |
| 需求优先级排序 | 产品经理 | 开发者 |

### 运维线

| 场景 | 归谁 | 不归谁 |
|------|------|--------|
| ECS服务器Docker/容器/端口 | ECS运维 | 本地运维 |
| 云电脑装软件/依赖/进程 | 本地运维 | ECS运维 |
| 云电脑外部可达性（隧道/反代） | ECS运维 | 本地运维 |
| 部署到服务器 | ECS运维 | 系统开发者 |

### 创作线

| 场景 | 归谁 | 不归谁 |
|------|------|--------|
| 表情包/插画设计产出 | 动画设计师 | 动画导演 |
| 分镜脚本/导演规划 | 动画导演 | 动画设计师 |
| 游戏美术 | 游戏美术设计师 | 动画设计师 |
| 混音/母带处理 | 混音母带工程师 | 系统开发者 |
| 小说创作 | 网文写手 | 动画导演 |

### 运营线

| 场景 | 归谁 | 不归谁 |
|------|------|--------|
| 小红书/微博/知乎图文 | 自媒体运营 | 短视频运营 |
| 抖音短视频 | 短视频运营 | 自媒体运营 |
| 素材获取 | 找对应设计师 | 运营自己生图 |

### 体系线

| 场景 | 归谁 | 不归谁 |
|------|------|--------|
| 改体系文件（规范/脚本/入口） | AI调教师 | 其他角色 |
| 改自己角色RULES/knowledge | 角色自己 | AI调教师 |
| 发现体系问题提bug单 | 角色→调教师 | 角色自己改 |
| 主人档案维护 | AI调教师 | 其他角色 |

## 常见协作链路

```
主人有想法 → 产品经理（翻译成PRD+验收标准）→ 通知开发者实现
                                             → 通知测试工程师按验收标准验收
系统开发者 → 写部署文档 → 交接台 → ECS运维领单部署
系统开发者 → 提交测试物料 → 交接台 → 测试工程师领单测试
测试工程师 → 发现Bug → 工单 → 对应开发者领单修复 → 修完提回归工单 → 测试回归验证
测试工程师 → 测试通过 → 工单 → 本地运维部署测试环境 → 主人验收
本地运维 → 主人验收通过 → 工单 → ECS运维正式部署上线
动画设计师 → 产出素材 → 交接台 → 自媒体运营领单分发
网文写手 → 完成小说 → 交接台 → 动画导演领单改编
产品经理 → 需求变更 → 通知所有相关角色 → 各角色调整
```
