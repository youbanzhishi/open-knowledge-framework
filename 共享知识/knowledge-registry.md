# 知识注册表

> 中央索引：记录所有角色的语言/技能知识位置，新角色创建时自动发现可引用的知识
> 最后更新：2026-05-10

## 使用方式

1. 新角色创建/入体系时，读此表
2. 对角色RULES.md中声明的每个技能标签，查此表
3. 发现已有知识 → 在角色knowledge/<语言>/下创建引用文件，指向持有角色的知识路径
4. 角色特有知识写在引用文件之外

## 维护规则

- 角色新增语言知识文件 → 更新此表对应行的"内容概要"
- 新角色注册技能标签 → 在此表追加行
- 新角色发现已有知识可引用 → 创建引用文件，不在此表重复注册
- 此表是追加不改写——新行追加，旧行更新内容概要

## 升级记录（knowledge → hot-rules）

> v2.0新增：记录knowledge升级为hot-rule的路径，可追溯、可降级

### 记录格式

| 升级时间 | 来源knowledge | 提炼为hot-rule | 触发次数 | 状态 | 备注 |
|----------|---------------|----------------|----------|------|------|
| - | - | - | - | - | - |

### 状态说明

| 状态 | 含义 |
|------|------|
| 活跃 | 热规则正在使用中 |
| 已降级 | 连续10次无违反，已退回knowledge |
| 已过期 | 规则已不适用，保留记录 |

### 维护规则

- knowledge升级为hot-rule → 在此表追加一行
- hot-rule降级 → 更新对应行的"状态"为"已降级"
- hot-rule内容变更 → 更新对应行的"提炼为hot-rule"列
- 此表是追加不改写——新记录追加，旧记录更新状态
